# Notion Work Log Skill

Skill for tracking daily work, accomplishments, and blockers in a Notion database.

## Usage
- Logging "What I did today".
- Tracking long-term project progress.
- Preparing for stand-ups.

## Template
**Date**: [YYYY-MM-DD]
**Task Category**: [e.g., Development, Bug Fix, Planning]
**Description**:
[A detailed log of work performed]

**Blockers/Notes**:
[Any issues encountered or things to follow up on]
